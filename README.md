## Employee Management System (GraphQL Backend)

This project is a backend Employee Management System developed using **Node.js**, **Express**, **GraphQL**, and **MongoDB (Atlas)**.  
It supports user authentication and full CRUD operations for employees, including image storage via **Cloudinary**.

---

## Technologies Used
- Node.js
- Express.js
- Apollo Server (GraphQL)
- MongoDB Atlas
- Mongoose
- JWT Authentication
- Cloudinary (Employee Profile Images)
- Postman (API Testing)

---

## GraphQL Endpoint
http://localhost:4000/graphql


---

## Implemented Features
User Signup
User Login (JWT-based authentication)
Add New Employee
Get All Employees
Search Employee by ID
Update Employee by ID
Delete Employee by ID
Search Employees by Designation or Department
Input validation and error handling
